# BYT Assignment 3 - JUnit Testing

Unit testing assignment for Money, Currency, Bank and Account classes using JUnit 4.

## How to Run Tests

### Running All Tests

**Linux/Mac:**
```bash
./gradlew test
```

**Windows:**
```bash
gradlew.bat test
```
